📄 HTML5란?

구조

HTML은 웹 페이지의 구조를 정의합니다. 시맨틱 태그를 사용하여 의미있는 마크업을 작성합니다.

<header>, <nav>, <section>, <article>, <footer>

요소

텍스트, 이미지, 링크, 폼 등 다양한 요소들을 포함할 수 있습니다.

<h1>~<h6>, <p>, <img>, <a>, <form>

의미

각 태그는 특정한 의미를 가지고 있어 검색 엔진과 스크린 리더에 도움이 됩니다.

SEO, 접근성 향상

🎨 CSS3란?

스타일

색상, 폰트, 레이아웃 등을 제어합니다.

color, font-size, background-color

레이아웃

Flexbox와 Grid를 사용한 현대적 레이아웃

display: flex, display: grid

애니메이션

부드러운 전환 효과와 애니메이션

transition, @keyframes, animation

⚙️ JavaScript란?

상호작용

사용자와의 상호작용을 처리합니다.

event listeners, DOM manipulation

동적 콘텐츠

런타임에 동적으로 콘텐츠를 생성합니다.

createElement, appendChild, innerHTML

데이터 처리

배열, 객체, 함수 등을 사용하여 데이터 처리

array methods, objects, async/await
